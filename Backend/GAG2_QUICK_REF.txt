# GAG2 Casino Quick Reference

✅ **Setup Complete! Database now has 49 GAG2 items**

## Quick Commands

### Import Items (if needed again)
```bash
npm run import-gag2
```
- Imports 16 pets + 33 seeds
- Automatically clears old GAG2 items
- Sets default values by rarity

### Give Items to Player
```bash
npm run auto-deposit -- <robloxId> "<item1,item2>" "<qty1,qty2>"
```

**Examples:**
```bash
# Give 1 Frog to player 123456
npm run auto-deposit -- 123456 "Frog" "1"

# Give mixed items
npm run auto-deposit -- 123456 "Frog,Apple,Fire Fern,Unicorn" "5,10,2,1"

# Give 100 Carrots  
npm run auto-deposit -- 123456 "Carrot" "100"
```

### Process Withdrawals
```bash
npm run auto-withdraw -- <robloxId> <quantity>
```

**Examples:**
```bash
# Process 5 items for player
npm run auto-withdraw -- 123456 5

# Process all 50 items
npm run auto-withdraw -- 123456 50
```
